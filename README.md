# Interior – Interior Design Landing Page

A modern, responsive **interior design landing page** built with **HTML, CSS, and JavaScript**. This project showcases a clean layout, smooth navigation, and visually appealing UI elements — perfect for interior design portfolios or personal landing pages.

---

## 🚀 Features

- **Responsive Design:** Works seamlessly on desktop, tablet, and mobile devices.  
- **Multiple Pages:** Includes Home, About, and Projects pages.  
- **Interactive Elements:** Smooth scrolling, hover effects, and clickable navigation.  
- **Portfolio Showcase:** Displays interior design projects with images and descriptions.  
- **Clean Code:** Well-structured HTML, CSS, and JS for easy customization.

---

## 🛠 Tech Stack

- **HTML5** – Semantic structure and content  
- **CSS3** – Flexbox, Grid, transitions, and responsive styling  
- **JavaScript** – Interactive features like smooth scroll and dynamic effects  
- **Images & Assets** – Optimized media for fast loading
